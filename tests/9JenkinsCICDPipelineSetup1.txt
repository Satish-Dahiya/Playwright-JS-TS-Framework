Steps to configure CICD Pipeline in Jenkins and its integration with Project Framework with Email/Slack Notification.

Download jenkins.war file from google and install it using 'java -jar jenkins.war' in the CMD window.
'Jenkins is fully up and running' message will be displayed in CMD.
Run jenkins on localhost:8080 and set username and password.

Create a Jenkinsfile in the root project and take its content from the Naveens github repo. (https://github.com/naveenanimation20/June2025PWFramework/)

In mean time , push the Jenkinsfile to the github repo.

New Item > Enter Item name as 'Playwright CICD Pipeline'
Select Pipeline
General > Enter Description
Pipeline > Select 'Pipeline script from SCM' in Definition.
SCM > select GIT 
Repository URL > Enter https://github.com/Satish-Dahiya/Playwright-JS-TS-Framework
Script Path > enter Jenkinsfile
Click Apply and Save

Run the Job/Pipeline from 'Build Now' button to trigger the build.
The look and feel / UI of the reports in the jenkins will be broken, so add the extenstion 'Disable Content-Security-Policy' in browser from google to fix it.
Any stage can be run once the job completes once from 'Restart' button.

Add the ESLint extension in VSC from Setting > Extensions, to get the errors in opend filed for any static code analysis errors.
